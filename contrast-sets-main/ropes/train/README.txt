Reasoning Over Paragraph Effects in Situations
----------------------------------------------------------------------------------------

This directory contains the ROPES dataset split into train and dev sets. The
test set is not included in this package. The format of each json file is the
same as that of SQuAD.

Please read the paper for more details:
https://www.semanticscholar.org/paper/Reasoning-Over-Paragraph-Effects-in-Situations-Lin-Tafjord/2ebb01d08022a52c1025302379873dedb5100035
